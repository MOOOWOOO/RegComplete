个人收集的常用正则表达式大全。

包含大小写字母和半角符号。

## 字母

包含大小写字母，不包含半角符号。

------------ | ------------- | ------------ | -------------

## IP

IPv4 地址, 包含 `0.0.0.0` 和 `255.255.255.255`

((?:(?:25[0-5]|2[0-4]\d|((1\d{2})|([1-9]?\d)))\.){3}(?:25[0-5]|2[0-4]\d|((1\d{2})|([1-9]?\d))))$

## 用户名

包含大小写字母数字和下划线，有限长度。

^[a-zA-Z]\w{`最小长度`, `最大长度`}$

## 密码

必须包含字母数字和特殊符号，有限长度。

^(?!\D+$)(?![^a-zA-Z]+$)\S{`最小长度`, `最大长度`}$
